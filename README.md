# KASPARRO

AI-powered SEO platform optimized for AI search engines like ChatGPT, Perplexity, and Gemini. Focuses on entity recognition, citation quality, and content understanding rather than traditional Google rankings.

## What is Kasparro?

Kasparro revolutionizes SEO by adapting to how AI search engines work. While traditional SEO tools chase Google rankings, Kasparro optimizes for AI comprehension - ensuring your brand is properly recognized, cited, and understood by AI systems.

## Key Features

- **AI-First Audit Pipeline**: 7 specialized modules analyzing entity recognition, sentiment, citation quality, EEAT signals, content relevance, brand context, and keyword coverage
- **Real-Time Dashboard**: Monitor brand performance across AI platforms with actionable metrics
- **Brand Optimization**: Get specific recommendations to improve your AI search visibility
- **Multi-Platform Support**: Optimized for ChatGPT, Perplexity, Gemini, and emerging AI search engines

## Why AI SEO Matters

AI search engines process information differently:
- **Entity Recognition**: How well AI systems identify your brand as a distinct entity
- **Citation Quality**: The credibility and context of your sources
- **Content Understanding**: How AI interprets and contextualizes your content
- **EEAT Signals**: Experience, Expertise, Authoritativeness, and Trustworthiness

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
npm start
```

## Usage

1. **Select Brand**: Choose from available brands in the dashboard
2. **Run Audit**: Execute comprehensive AI-SEO analysis
3. **Review Results**: Examine scores across all 7 audit modules
4. **Apply Recommendations**: Implement suggested optimizations
5. **Monitor Progress**: Track improvements over time

## Project Vision

Kasparro bridges the gap between traditional SEO and AI-driven search. As AI search engines become mainstream, brands need tools that speak their language - focusing on semantic understanding, entity relationships, and content quality over algorithmic ranking factors.
